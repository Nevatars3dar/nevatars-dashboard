## ğŸ› ï¸ Troubleshooting

### **"Error loading data" message**

**Problem:** Dashboard can't find `assets_data.json`

**Solution:**
1. Run "Sync Registry" in Maya first
2. Make sure `assets_data.json` exists in same folder as HTML
3. Refresh browser page

### **"Address already in use" error**

**Problem:** Port 8080 is already taken

**Solution:**
1. Close other applications using port 8080
2. OR edit `start_server.py` and change `PORT = 8080` to `PORT = 8081`

### **Team can't access via Network URL**

**Problem:** Firewall blocking port 8080

**Solution:**
1. Windows Firewall â†’ Allow app through firewall
2. Add Python to allowed apps
3. Enable for Private networks

### **Dashboard loads but no data shows**

**Problem:** Browser cache or old JSON file

**Solution:**
1. Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
2. Check `assets_data.json` was generated recently
3. Re-run "Sync Registry" in Maya
